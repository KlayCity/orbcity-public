# orbcity-contracts

# Quick start

First, install library

```sh
npm install
```

Copy .env_sample to .env and write your information.

Compile:

```sh
npm run compile
```

Run test:

```sh
npm run test ./test/Basic.test.ts
```

Run script to testnet:

```sh
npm run testnet ./scripts/deploy-yarn.ts
```

Run script to mainnet:

```sh
npm run testnet ./scripts/deploy-yarn.ts
```
